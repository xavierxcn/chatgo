# chatgo
chatgpt cmd tool

## Usage

```bash
# install
$ go install github.com/xavierxcn/chatgo@latest

# set openai token
$ chatgo set <your-token>

# chat
$ chatgo chat

init robot...
init robot success.
> 你叫什么名字
chatgo: 
我叫ChatGo，很高兴能够为你服务！有什么可以帮到你的吗？


> 


```
